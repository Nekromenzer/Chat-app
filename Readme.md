<!-- project setup -->

Front-end (client)

1.  cd client
2.  npm install
3.  npm start
(front-end app will start in localhost:3000)

Back-end (server)

1.  cd server
2.  npm install
3.  npm start
(back end app will start at localhost:5000)

You should start both client and server functioning of app!
